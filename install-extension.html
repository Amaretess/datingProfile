How to install and use visual studio code Live Server Extension

Install | Use | Tutorial

Installation instructions

  1. Visual studio code In vs code, click the extension icons in the left side of the page
  2. In search bar type LIVE SERVER
  3. select live server extension, it should be top choice
  4. click the green install button

Usage instructions

  • vs code must be open for live server to work
  • right click code and select open with live server
